# fie-report


---

retcode 日志上报模块

## API


| 成员        | 说明           | 类型               | 默认值       |
|-------------|----------------|--------------------|--------------|
|coreCommand      | 核心模块使用的日志上报方法   | Method |       |
|moduleUsage      | 套件／插件使用的日志上报方法   | Method |       |
|error      | 自定义retcode发送日志   | Method |       |
