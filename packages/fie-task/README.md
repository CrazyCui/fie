fie 数据缓存模块, 可以用来存储用户常用数据,支持有效期设置

## 安装

```
npm install fie-task --save
```

## API

### has(tasks, when)

> 是否存在当前时机的任务流

- tasks `{array}` 任务列表
- when `{string}` 时机
